## Rodar local

### Instalar dependências do Serverless Framework globalmente

```shell
npm i -g serverless serverless-dynamodb-local serverless-offline
```

### Instalar Dynamodb localmente

```shell
sls dynamodb install
```

### Iniciar servidor local 

```shell
sls offline start
```
